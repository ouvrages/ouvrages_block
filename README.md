# OuvragesBlock

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ouvrages_block'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ouvrages_block

## Standard blocks

- standard_block_title
- standard_block_image
- standard_block_rich_text
- standard_block_medium_collection
